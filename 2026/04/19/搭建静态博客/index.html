5.3 GitHub Token 认证

GitHub 不再支持密码推送，需使用 Personal Access Token：

  1. 访问 https://github.com/settings/tokens
  2. Generate new token (classic)
  3. 勾选 repo 权限
  4. 复制 token（只显示一次！）
  5. 推送时用户名填 GitHub 用户名，密码填 token
